# COVID-19 Data Analysis in South Korea, January-June 2020

## Overview
This repository contains an exploratory data analysis (EDA) of COVID-19 data in South Korea. The analysis aims to provide insights into the spread of the virus, its impact, and trends over time, creating a jumping off point for policy recommendations for future pandemics.

## Data Source
The data used in this analysis is sourced from the [Kaggle COVID-19 dataset](https://www.kaggle.com/kimjihoo/coronavirusdataset) maintained by Kim Jihoo. The dataset includes information about COVID-19 cases in South Korea, including the number of confirmed cases, deaths, recoveries, outbreak locations and sizes, and demographic information.

## Analysis Contents
- **Data Cleaning**: Preprocessing steps taken to clean and prepare the data for analysis.
- **Exploratory Data Analysis (EDA)**: Analysis of COVID-19 trends, including:
  - Cumulative cases
  - Case distribution by age and gender
  - Mortality rates
- **Visualization**: Visual representations of key findings using plots and graphs.

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Folium

## Conclusion
This analysis takes a particular focus on the social, political, and health ramifications of the Shincheonji Church outbreak. Recommendations include a focus on social health.
